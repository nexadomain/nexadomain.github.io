About

Learn more about WhatsJhom

WhatsJhom is a Web Application which let people initiate the WhatsApp Chat without Saving anyone phone number. Using WhatsJhom, you can save plenty of Phone Memory without saving anyone Phone Number. WhatsJhom is free to use and it can be used without Internet just visit WhatsJhom Website (www.jhom.co) once and use it without Internet for a year.
